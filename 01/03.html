1-1)¿Que es CSS?

las siglas css son el nombre de una de las tres tecnologías que hacen la WEB. CSS significa Hojas de Estilo en Cascada en inglés Cascading Style Sheets y te dará los estilos (diseño) necesarios para que un sitio web o una aplicación web se vean visualmente bien.

1-2)

CSS se puede usar, por ejemplo, para cambiar la fuente, el color, el tamaño y el espaciado del contenido, para formar multiples columnas, añadir animaciones y otros elementos decorativos.

1-3)

La meta básica del lenguaje Cascading Stylesheet (CSS) es permitir al motor del navegador pintar elementos de la página con características específicas, como colores, posición o decoración

1-4)Cuál es la anatomía de la regla de CSS? Agregar imagen explicativa y utilizando una lista descriptiva explicar sus elementos.

Una regla de CSS está formada por una parte llamada “selector” y otra parte llamada “declaración” (puede haber una o varias dentro del mismo selector).

1-5) que es un selector de CSS?

El elemento HTML en el que comienza la regla. Esta selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos p). Para dar estilo a un elemento diferente, solo cambia el selector.

1-6)

tipos de selectores:

  • Selector de id:

    El elemento en la página con el ID especificado (en una página HTML dada, solo se permite un único elemento por ID)

  • Selector de clases:

    Los elementos en la página con la clase especificada (una clase puede aparecer varias veces en una página).

  • selector universal:

    Se utiliza para seleccionar todos los elementos de la página

  • selector de etiquetas:

    Todos los elementos HTML del tipo especificado.

  • selector descendiente:

    El "␣" combinador (que se supone que representan un espacio, o mejor dicho uno o más espacios en blanco) combina dos selectores tales que el selector combinado incluye sólo los elementos que coinciden con el segundo selector para los que hay un elemento ancestro que coincide con el primer selector. Los selectores descendientes son similares a selectores hijos , pero que no requieren que la relación entre los elementos coincidentes ser estrictamente entre padres e hijos.
